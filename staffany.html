Staffany

Company: Staffany is a company based in Singapore that creates a software to help make scheduling employees easier.

Goal: Create a branding and user testing on the software

Role: UX/UI Designer


Findings

- Research -

Research was needed when conducting the company's logo/tagline and creating the style guide, such as understanding color theories, font representation for the company, and assets. We decided on three primary colors with each color containing its own meaning: Medium Turquoise, Froly, and Grandis. Our branding style guide consisted of what the company is about, correct/incorrect logo usage, logo sizes, primary/secondary colors, and correct/incorrect text usages.





- User Flow/Competitive Analysis -

My team and I conducted research based on competitors to get a better engagement with the user process and understanding the competitor market.




- Wireframes -

I iterated many different versions of wireframes.






- User Testing -

My team and I conducted user testing and gathered feedback from the user. Firstly, I sent an email notifying the user the procedure of what to expect. I asked the user 3 questions before engaging them in the testing application process. I then gave the user step-by-step instructions on what to do without providing the user an answer. When the testing was finished, I reviewed the recorded videos to make more detailed notes and analyzing every mouse movement/thought process the user was making. I then wrote a deep-analysis written report for the team on what the feedback was and how to improve. *Due to privacy reasons, I will not release user testing videos.*
